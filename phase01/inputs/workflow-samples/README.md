# inputs/workflow-samples/ （强烈建议）

放 **3~5+ 个真实开源项目**的 `.github/workflows/*.yml`，作为真实负载。

请补充：
- 按项目分子目录，如 `projectA/ci.yml`
- 覆盖多样场景：构建 / 测试 / 发布 / matrix / reusable workflow / cache / 部署
- 附 `SOURCES.md`：每个样本的来源仓库、选取理由

消费方：compat-diff · usability · L0 冒烟
缺失影响：差异发现偏理论、迁移摩擦无从实测。

> 补充完成后可在此文件末尾记：`已补充 / 样本数 / 日期`。
