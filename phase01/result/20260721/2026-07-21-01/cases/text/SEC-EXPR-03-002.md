标题:      uses 字段应支持 ${{ }} 表达式求值

前置条件:
  - 存在 push 触发的 workflow

操作步骤:
  1. 在 uses 字段中使用 ${{ atomgit.repository }} 等表达式
  2. 观察是否正确求值并引用 action
  3. 若平台不支持，验证错误信息质量

预期结果:
  - uses 字段中的 ${{ }} 表达式应被正确求值
  - 若不支持，应产生明确的错误信息说明

验证点:
  - [正向] uses 中表达式求值正确或产生明确错误
  - [负向] 不应静默失败

清理:      none
